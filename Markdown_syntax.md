# 1. How to use MarkDown syntax
## 1.1. 헤더

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```
* 사용 예
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

## 1.2. 강조
```
*이탤릭체* or _이탤릭체_
**볼드체** or __볼드체__
~~취소선~~
```
* 사용 예   
*이탤릭체*
**볼드체**
~~취소선~~

## 1.3. 리스트

```
1) 번호 x (모두 같은 결과)
* 리스트 1
- 리스트 2
+ 리스트 3

2) 번호 o
1. 리스트 1
2. 리스트 2
3. 리스트 3
```
* 사용 예
1. 리스트 1
2. 리스트 2
3. 리스트 3

## 1.4. 수평선
```
* * *
- - -
```
* 사용 예
* * *
- - -

## 1.6. 인용문
```
> 인용문입니다.
>> 인용문 안 인용문입니다.
>>> 인용문 안 인용문 안 인용문입니다.
```
* 사용 예

>> 인용문 안 인용문입니다.

## 1.7. 표
```
(오류 시, 테이블 앞을 줄바꿈 처리)
| 왼쪽정렬 | 가운데정렬 | 오른쪽정렬 |
| :--- | :---: | ---: |
| 심교훈 | 수학 | 85 |
| 문태호 | 영어 | 90 |
| 황병일 | 국어 | 95 |
```
* 사용 예   

| 왼쪽정렬 | 가운데정렬 | 오른쪽정렬 |

| 심교훈 | 수학 | 85 |
| 문태호 | 영어 | 90 |
