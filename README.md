# 1. How to use MarkDown syntax
## 1.1. 헤더(Headers)

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```

* 사용 예
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

## 1.2. 리스트(list)

```
1) 번호 x
* 리스트 1
- 리스트 2
+ 리스트 3

2) 번호 o
1. 리스트 1
2. 리스트 2
3. 리스트 3
```

* 사용 예
* 리스트 1
- 리스트 2
+ 리스트 3
1. 리스트 1
2. 리스트 2
3. 리스트 3
