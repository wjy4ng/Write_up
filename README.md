# 1. How to use MarkDown syntax
## 1.1. 헤더(Headers)

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```

* 사용 예
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

## 1.2 강조
```
*이탤릭체* or _이탤릭체_
**볼드체** or __볼드체__
~~취소선~~
<u>밑줄</u>
<mark>형광펜</mark>
```

* 사용 예
*이탤릭체*
**볼드체**
~~취소선~~
<u>밑줄</u>
<mark>형광펜</mark>

## 1.3. 리스트(list)

```
1) 번호 x (모두 같은 결과)
* 리스트 1
- 리스트 2
+ 리스트 3

2) 번호 o
1. 리스트 1
2. 리스트 2
3. 리스트 3
```

* 사용 예
1. 리스트 1
2. 리스트 2
3. 리스트 3
